Carte Mentale

Carte Mentale Interactive

La carte mentale visualise les connexions conceptuelles entre les contributions. Elle révèle comment des éléments culturels différents convergent autour de concepts universels.

Concepts clés révélés

Par exemple, pour “Nos Rituels de Fin d’Année” :

  • Convivialité : Réveillon (FR), Christmas Dinner (UK), Célébration tropicale (MQ)
  • Tradition Familiale : Présence, partage, transmission
  • Rupture avec le Quotidien : Préparation spéciale, vêtements, rituels distincts
  • Nourriture comme Vecteur : Omniprésent dans toutes les cultures

Comment l’utiliser ?

  1. Cliquez sur un concept central
  2. Explorez les branches (pays/contributions)
  3. Découvrez les similarités universelles
  4. Identifiez les nuances culturelles qui les distinguent

Pédagogie

La carte mentale transforme les observations en analyse structurée. Elle aide les élèves à comprendre que la diversité culturelle repose souvent sur des fondamentaux humains partagés.
