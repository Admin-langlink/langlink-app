Mosaïque Culturelle

La mosaïque est la vue d’ensemble visuelle de toutes les contributions. Chaque brique représente une contribution d’un élève, reliée thématiquement aux autres.

Comment ça fonctionne ?

  • Zoom sur les briques : Cliquez sur une contribution pour lire l’analyse complète
  • Filtrer par thème : Explorez les “Fêtes”, “Rituels”, “Traditions”
  • Filtrer par pays : Découvrez les contributions d’une culture spécifique
  • Voir les connexions : Les lignes relient les contributions similaires ou complémentaires
